## About This Project

Welcome to the repository for my personal portfolio, a full-stack application I built to showcase my work in data strategy, analytics, and leadership. This site serves as a central hub for my professional accomplishments and features detailed case studies of my most impactful projects. 

While I am a data leader by trade, I believe in being hands-on with technology. This project was an opportunity to build a professional platform from the ground up using a modern, performant web stack.

## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You will need to have Node.js (v18 or later) and npm (or your preferred package manager) installed.

### Installation & Setup

1.  **Clone the repository:**
    ```sh
    git clone [https://github.com/Derek-Devs/derekdevs.git](https://github.com/Derek-Devs/derekdevs.git)
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd derekdevs
    ```
3.  **Install dependencies:**
    ```sh
    npm install
    ```

### Running the Development Server

1.  **Start the dev server:**
    ```sh
    npm run dev
    ```
2.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
